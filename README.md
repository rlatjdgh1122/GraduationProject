# 🎮 게임 소개

# **컨셉**

게임의 컨셉은 “**적들을 처치하며 나만의 왕국을 건설해 나가자” 입니다.**

# 플레이 방식

플레이어는 3칸의 빙하를 가지고 시작합니다. 
적들의 빙하가 플레이어의 빙하에 부딪히면 전투를 하게됩니다.  

플레이어는 펭귄 집합소를 통해 펭귄을 구매하고 군단에 배치할 수 있습니다.  

또한 넥서스를 통해 여러 디펜스 건물을 설치해 게임을 즐길 수 있습니다.
